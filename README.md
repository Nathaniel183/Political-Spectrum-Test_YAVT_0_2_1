# YAVT 0.2.1 · 五轴政治倾向测试（本地版）

直接下载HTML在浏览器打开就可以测试

一个**纯前端、可离线运行**的 YAVT（Yet Another Values Test）实现，基于 **YAVT 0.2.1** 题库与 **五轴模型**。适合直接部署在 GitHub Pages，或下载到本地浏览器中使用。

> ⚠️ 本测试仅用于**价值取向的自我探索**，不构成任何政治立场或现实判断。

---

## ✨ 特点

* 📱 **移动端优先**：纵向布局，适合手机浏览
* 🌱 **纯前端实现**：单个 HTML 文件，无需后端
* 🎨 **莫兰迪配色**：简洁、克制、美观
* 🧮 **算法透明**：完全按照公开 JSON 与公式计算
* 🧭 **五维坐标轴**：直观展示政治价值分布

---

## 🧭 五个坐标轴（Dimensions）

| 轴 | 左端   | 右端   |
| - | ---- | ---- |
| 1 | 计划经济 | 市场经济 |
| 2 | 世界主义 | 国家主义 |
| 3 | 自由   | 权威   |
| 4 | 进步   | 守成   |
| 5 | 环保   | 发展   |

每个轴最终得分标准化到 **0–100**。

---

## 📝 题目与作答方式

* 题库版本：**YAVT 0.2.1**
* 每道题对应五个轴的权重向量 `evaluation`
* 作答选项与权重：

| 选项       | 权重   |
| -------- | ---- |
| 强烈同意     | +1   |
| 同意       | +0.5 |
| 中立 / 不确定 | 0    |
| 反对       | -0.5 |
| 强烈反对     | -1   |

每道题对各轴的贡献为：

```
轴增量 = evaluation[i] × 选项权重
```

---

## 📐 得分计算方式

1. **预计算**：

   * 对每个轴，计算所有题目在“强烈同意 / 强烈反对”情况下的 **最大可能绝对值和**

2. **答题完成后**：

   * 累加用户在每个轴上的加权得分

3. **标准化**：

```
轴得分 = 50 + (该轴得分 / 该轴最大绝对值和) × 50
```

* 得分范围：`0 – 100`
* 中立点：`50`

最终得到一个五维向量，例如：

```
[72, 20, 27, 40, 61]
```

---

## 🏷️ 解释标签（Explanations）

每个轴都被平均划分为 **7 个区间**，对应：

```
[极左] [偏左] [略左] [中立] [略右] [偏右] [极右]
```

程序根据轴得分所在区间，显示对应的文字标签（如「偏向平等」「民族主义」等）。

---

## 🧠 意识形态判定（Ideologies）

* 每种意识形态在五轴空间中是一个 **固定点**（0–100）
* 用户得分向量与所有意识形态点计算 **欧式距离**

```
d = √Σ(用户轴得分 − 意识形态轴值)²
```

* 距离 **最近** 的意识形态即为最终结果

示例：

```
用户得分 → [72,20,27,40,61]
最近意识形态 → 马克思列宁主义
```
